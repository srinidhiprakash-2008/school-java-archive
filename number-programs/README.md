# Number Programs

A collection of Java programs focused on numerical operations, mathematical logic, and problem-solving.

These programs were developed during my school years while learning core programming concepts such as loops, conditional statements, functions, and arithmetic operations.

Many of these exercises served as an introduction to algorithmic thinking and helped build a strong foundation in logical reasoning.

## Topics Covered

Examples include:

* Armstrong Numbers
* Palindrome Numbers
* Prime Numbers
* Reverse Numbers
* Sum of Digits
* Factorials
* Fibonacci Series
* Number Conversions
* Perfect Numbers
* Mathematical Utilities

## Purpose

This collection was created to practice numerical problem-solving and strengthen understanding of programming fundamentals through hands-on implementation.
