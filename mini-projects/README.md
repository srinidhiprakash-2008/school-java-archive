# Mini Projects

A collection of small Java applications developed during my school years using BlueJ.

Unlike the programs in other categories, these projects combine multiple programming concepts such as user input, decision making, loops, functions, and basic logic to create complete, interactive applications.

These projects represent an early transition from solving individual programming exercises to building programs that simulate real-world tasks and user interactions.

## Project Types

Examples may include:

* Calculator Applications
* Charity Management Systems
* Student Record Management
* Menu-Driven Utilities
* Basic Data Processing Programs
* Interactive Console Applications

## Purpose

These projects were created to strengthen problem-solving skills, program structure, and user interaction design while learning Java fundamentals.
